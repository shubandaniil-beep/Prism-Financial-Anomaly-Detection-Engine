# Prism-Financial-Anomaly-Detection-Engine
PRISM — движок выявления финансовых аномалий. Спектральный анализ (FFT), сезонная декомпозиция и ансамбль 6 детекторов (EWMA, CUSUM, Hampel медиана, MAD, Isolation Forest) выявляют всплески и сдвиги в потоках биржи и банка. Каждая находка получает оценку 0–100%, вклады и объяснение через Claude API. Поддержка Binance, CSV, батч и потокового режима.
